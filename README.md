FlyteCTL is designed to be a portable and lightweight CLI to work with Flyte. It is written in Golang and can access [FlyteAdmin](https://github.com/flyteorg/flyteadmin/)‚Äîthe control plane of Flyte.

## üöÄ Quick Start

* Install FlyteCTL with bash or shell script

    * Bash
        ```bash
        $ brew install flyteorg/homebrew-tap/flytectl

        # Upgrade flytectl 
        $ brew upgrade flytectl
        ```
    * Shell script
        ```bash
        $ curl -s https://raw.githubusercontent.com/lyft/flytectl/master/install.sh | bash
        ```

* Start sandbox using FlyteCTL 
    ```bash
    $ flytectl sandbox start 
    ```

* Register Examples
    ```bash
    # Register core workflows 
    $ flytectl register examples -d development -p flytesnacks
    ```
